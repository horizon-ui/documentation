CloseButton is essentially a button with a close icon. It is used to handle the
close functionality in feedback and overlay components like
[Alerts](/docs-horizon-ui-dashboard/docs/feedback/alert),
[Toasts](/docs-horizon-ui-dashboard/docs/feedback/toast),
[Drawers](/docs-horizon-ui-dashboard/docs/overlay/drawer) and
[Modals](/docs-horizon-ui-dashboard/docs/overlay/modal).

## Import

```js
import { CloseButton } from "@chakra-ui/react"
```

## Usage

```jsx
<CloseButton />
```

```tsx
<CloseButton />
```

### Button Size

Pass the `size` prop to adjust the size of the close button. Values can be `sm`,
`md` or `lg`.

```jsx
<Stack direction="row" spacing={6}>
  <CloseButton size="sm" />
  <CloseButton size="md" />
  <CloseButton size="lg" />
</Stack>
```

```tsx
<Stack direction="row" spacing={6}>
  <CloseButton size="sm" />
  <CloseButton size="md" />
  <CloseButton size="lg" />
</Stack>
```
